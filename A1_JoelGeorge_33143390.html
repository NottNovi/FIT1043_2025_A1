A3. Supervised Learning¶

Question 1¶

Supervised machine learning is a process where you teach a model to make predictions by showing it examples with correct answers. Usin the current dataset the predictive model will be to use the Age, StudyTimeWeekly, Absences, and ParentalSupport to determine the predcted Grade Class

The features of Machine Learning are as follows: input- data you feed the model, such as the Age, StudyTimeWeekly, Absences, and ParentalSupport label- the answer that you want the model to prdict (GradeClass)

From this the given input the model is able to take this in and give a prediction on the grade class.

Training vs. Testing Training Set: This is most of the dataset and it is used to teach the model. The model looks at the features and the corresponding GradeClass labels in this set over and over, adjusting its internal logic to find the relationships. Test Set: This is a smaller part of your data that is kept hidden from your model during training. After the model is trained, you use this set to evaluate its performance on data it has never seen before.

Risk of Not Using a Test Set The risk of not using a test set is overfitting. Overfitting occurs when a model learns the training data too well and memorizes the specific examples and their random noise instead of learning the patterns. An overfitted model will be hiighly accurate on the data it was trained on but will fail when it encounters new, real world data. Without a test set, a flase confidence on the real world data will be displayed.
